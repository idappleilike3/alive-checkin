1. 第一次設定守護人

第一次進入本服務時,系統會要求你設定至少 1 位守護人。這是產品核心安全設計 — 沒有守護人,就無法在緊急時通知任何人。

為什麼需要守護人?

當你未按時簽到、SOS 緊急求助、或長時間失聯時,系統會主動聯絡你的守護人。他們會收到 LINE 訊息,並附上你最後一次的位置(若有開啟定位)。

建議選誰?

  • 家人(爸媽、配偶、兄弟姊妹)
  • 同居人 / 室友
  • 親近的朋友
  • 同事(白天可聯絡的對象)
守護人不需下載任何 App,也不需註冊。他們只需要有 LINE,並能接收你的訊息。

至少填哪些資料?

  • 姓名(*必填)
  • 關係(*必填,例如「家人」、「朋友」)
  • 手機號碼 或 Email(*至少填一個)
  • 是否為主要聯絡人(可選,第一位會自動設為主要)

守護人會收到什麼?

守護人會在你未簽到、SOS、或其他異常狀況發生時,收到 LINE 訊息。訊息內容包含你的姓名、時間、位置連結(若有)。他們不需要做任何操作,只是「被通知」。

如何解除綁定?

進入「守護群」分頁 → 找到該守護人 → 點「刪除」→ 確認。系統會立即停止未來對此人的通知,但已發送的訊息無法收回。

2. 每日如何簽到

首頁有醒目的「今日簽到」按鈕。點下去就完成,不用打字。

  1. 進入首頁
  2. 找到「今日簽到」按鈕(主要操作區)
  3. 點按鈕 → 自動記錄今天的平安狀態
  4. 按鈕變成「今日已簽到」完成狀態
每天只算一次。同一日重複點按會顯示提示,不會重複計算連續天數。

忘記簽到怎麼辦?

不用擔心,首頁有「補簽現在」按鈕(在設定區內),可以補登今天或昨天。但連續天數可能因此中斷。

3. 如何新增或更換守護人

進入底部導覽「守護群」分頁,可以看到所有守護人列表。

新增守護人

  1. 進入「守護群」分頁
  2. 點「新增守護人」
  3. 填寫姓名、關係、手機 / Email
  4. 選擇「設為主要」(選填,第一位會自動設為主要)
  5. 點「儲存」

編輯守護人

在守護人項目點「編輯」按鈕,即可修改資料。設定為「主要」會自動取消其他人的主要狀態。

刪除守護人

點「刪除」按鈕 → 系統會跳確認視窗 → 確認後刪除。已發送的歷史訊息不會收回。

邀請好友成為守護人(透過 LINE)

點「生成邀請連結」,複製 LINE 訊息連結,傳給對方。對方開啟後,系統會引導完成綁定。

4. 如何設定提醒時間

在「我的設定」分頁,可以設定每日提醒簽到的時間(例如每天 09:00)。

提醒時間僅在「付費方案」生效。免費版只有當日未簽到時,在網頁內顯示提醒。

提醒方式:

  • 網頁內提醒(免費版可用)
  • LINE 推播提醒(付費版可用)
  • Email 提醒(付費版可用)

5. 沒有簽到時會發生什麼

系統會依「寬限時間」分階段通知:

  1. 超過寬限時間 → 發送預警 LINE 給你(本人)
  2. 預警後 15 分鐘無回應 → 通知核心守護人(主要聯絡人)
  3. 若仍無回應 → 升級至所有守護人(799 方案)
這不是救護系統。若有立即危險,請撥打 119 或 110。

6. SOS 緊急求助如何使用

首頁右下角紅色 SOS 浮動按鈕。

  1. 點 SOS 按鈕(需二次確認)
  2. 確認 → 立即通知所有核心守護人
  3. 若已開啟定位,會附上你的最後位置
重要:SOS 是通知功能,不是報警系統
若有立即危險(火災、暴力、醫療緊急),請:
• 撥打 119(消防/救護)
• 撥打 110(警察)
• 撥打 112(手機無訊號時緊急號碼)

7. 如何修改會員資料

在「會員中心」分頁,可以查看:

  • 目前方案與到期日
  • 守護人使用數 / 上限
  • 提醒服務狀態
  • 付款紀錄入口
  • 客服聯絡方式

如需升級方案、取消續訂、刪除帳號,請聯絡客服。

8. 常見問題

Q:忘記簽到會怎樣?

會依寬限時間分階段通知守護人,請盡量記得每天簽到。可在「我的設定」調寬限時間。

Q:可以不要填滿所有守護人名額嗎?

可以。免費版只要 1 位守護人,399 方案最多 3 位,799 方案最多 5 位。系統不會強迫填滿,只會禮貌提醒。

Q:守護人不會用 LINE 怎麼辦?

守護人必須能用 LINE 才能接收通知。但 Email 是備用方案,系統會在 LINE 推送失敗時寄 Email(付費版)。

Q:可以取消訂閱嗎?

可以。在「會員中心」找到「取消方案」,或聯絡客服。我們不會設下任何「找不到的取消按鈕」陷阱。

Q:資料會被刪除嗎?

不會主動刪除。你可以透過「會員中心 → 刪除帳號」主動要求。我們會在 30 天後真正刪除(留緩衝期讓你反悔)。

Q:可以匯出資料嗎?

可以。「會員中心 → 我的資料管理 → 匯出我的資料」會把簽到紀錄、守護人資料、通知紀錄打包成 JSON 檔給你下載。

9. 隱私與資料安全

我們蒐集什麼?

  • 你的 LINE User ID(用於推播)
  • 每日簽到時間紀錄
  • 守護人資料(姓名、關係、聯絡方式)
  • 你的位置資訊(只在你開啟定位時記錄)

我們不蒐集什麼?

  • 你的 LINE 對話內容(只記錄「簽到」動作)
  • 你的瀏覽紀錄或裝置資訊(超過必要範圍)
  • 你的朋友清單

誰能看到我的資料?

  • 只有你本人(透過 LINE 帳號驗證)
  • 你設定的守護人(只在他們收到通知時)
  • 我們的後台管理員(僅在客服需求時,需雙重驗證)

資料儲存位置?

主要儲存在台灣的雲端資料庫,符合台灣個資法規範。我們不使用中國或俄羅斯等高風險地區的服務。

出了事怎麼辦?

若發生資料外洩事件,我們會在 72 小時內通知所有受影響用戶,並主動報告主管機關。
