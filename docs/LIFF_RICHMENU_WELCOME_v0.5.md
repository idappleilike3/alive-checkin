### 互動設計

1. **頁面載入** → 開始 5 秒倒數
2. **倒數 0 → 自動送出** → 跳轉「已發出 SOS」頁
3. **倒數中按 [取消]** → 立即取消,跳回首頁
4. **倒數中按 [確認發出]** → 立即送出,跳轉「已發出 SOS」頁
