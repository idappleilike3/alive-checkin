# 註冊引導、提醒與 LINE 圖文選單設計規格

日期：2026-07-20  
狀態：已完成需求討論，等待文件確認後實作

## 1. 本次確定修改範圍

- 首頁主按鈕「我還活著」統一改為「我已平安」。
- 逾期狀態文字改為白色，確保深色與警示底色上的辨識度。
- 新會員必須完成至少 1 位守護人的資料、LINE 邀請、對方同意與提醒時間設定，才能進入主要功能。
- 註冊尚未完成時仍保留 SOS 與撥打 119，不阻擋緊急求助。
- 守護人關係採下拉選單；選擇「其他」才顯示自行輸入欄位。
- 新會員設定守護人後，立即進入提醒時間設定，不必停在畫面等待對方同意。
- 對方尚未同意時顯示「等待守護人同意中」；同意後通知會員並開放首頁。
- 提醒時間支援依方案新增多個時段，不能超過方案上限。
- 已有 1 位守護人但尚未填滿方案名額時，預設每週提醒一次；可選立即新增、明天提醒、每週提醒或暫停提醒。
- 採用 A 版 6 格大字圖文選單，重新製作高對比、高質感版本。
- 新增 LINE 加好友歡迎訊息與「開始設定守護人」大字綠色按鈕。
- 歡迎訊息補上「完成設定即享 7 天免費安心體驗」。
- 採用 799 B 方案：月費可建立 1 個守護群，年費可建立 3 個守護群。
- 取消全部電話通知權益與文案。
- 取消專屬客服、優先客服權益與文案。

## 2. 註冊引導流程

1. 使用者加入 LINE 官方帳號並以 LIFF 一鍵登入。
2. Bot 傳送歡迎訊息，主按鈕為「開始設定守護人」。
3. 使用者填寫守護人姓名、關係與電話。
4. 系統產生 LINE 邀請並讓使用者直接分享。
5. 使用者接著設定每日簽到提醒時間。
6. 系統顯示守護人「等待同意」狀態。
7. 守護人透過 LINE 同意後，系統發送雙方確認訊息並開放首頁。

註冊完成條件：至少 1 位守護人狀態為已同意，且至少設定 1 個提醒時間。只有填資料或送出邀請，不視為完成。

## 4. 每日提醒規則

| 方案 | 每日提醒上限 | 建議初始時段 |
|---|---:|---|
| 免費會員／7 天體驗 | 1 次 | 12:00 |
| 199 月費／年費 | 2 次 | 12:00、18:00 |
| 399 月費／年費 | 2 次 | 12:00、18:00 |
| 799 月費／年費 | 3 次 | 09:00、12:00、18:00 |

- 初始時段會預先帶入，但使用者必須在註冊流程中確認，可自行更改。
- 後端依方案驗證提醒數量，前端隱藏或停用超出上限的新增按鈕。
- 每個提醒時段每日最多發送一次；完成當日簽到後，當天其餘提醒全部停止。

## 6. A 版 LINE 圖文選單

採 2500 × 1686 的 3 欄 × 2 列版型：

| 位置 | 按鈕 | 功能 |
|---|---|---|
| 左上 | 我已平安 | 開啟 LIFF 並完成今日簽到 |
| 中上 | SOS 求助 | 開啟 SOS 確認頁，不直接誤發 |
| 右上 | 綁定守護人 | 開啟守護人管理與邀請頁 |
| 左下 | 我的狀態 | 查看簽到紀錄、提醒與會員方案 |
| 中下 | 方案權益 | 查看月費、年費與升級方案 |
| 右下 | 設定教學 | 查看提醒、定位、守護人與守護群教學 |

視覺規格：

- 暖白背景、深藍黑文字，避免低對比灰字。
- 「我已平安」使用高彩度翡翠綠；「SOS 求助」使用鮮明紅色。
- 「方案權益」以金黃色作重點，其餘功能維持明亮白底。
- 每格同時使用圖示與大字粗體，不只依靠顏色辨識。
- 文字以長輩不需放大螢幕即可辨識為基準。
- 整體使用乾淨線條與少量手繪溫度，不使用灰暗底色、過細字體或複雜裝飾。

## 9. 驗收條件

- 新會員無法略過第一位守護人與提醒時間設定。
- 只有守護人完成 LINE 同意後，會員才可進入首頁；SOS 與 119 不受限制。
- 各方案只能設定允許數量的提醒時段，排程不重複發送。
- 799 月費可綁定 1 個守護群，年費可綁定 3 個守護群。
- 全站不再出現電話通知、專屬客服或優先客服權益。
- 歡迎訊息包含 7 天免費安心體驗與「開始設定守護人」按鈕。
- 圖文選單六格均能開啟正確頁面，文字在手機 LINE 內清楚可讀。
- 桌機與手機版均不得發生文字重疊、截斷或低對比問題。
