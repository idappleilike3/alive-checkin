## 📂 程式碼整合建議

未來在 `app.py` 可以這樣用:

```python
I18N_STRINGS = {
    "welcome": {
        "zh-TW": "🛡️ 歡迎來到「今天還在嗎」...",
        "en": "🛡️ Welcome to ...",
        "ja": "🛡️ 「今日も元気?」へようこそ...",
        "vi": "🛡️ Chào mừng đến với ...",
        "id": "🛡️ Selamat datang di ...",
    },
    "checkin_success": {...},
    "alert_wave1": {...},
    ...
}

def get_string(key: str, lang: str = "zh-TW") -> str:
    return I18N_STRINGS.get(key, {}).get(lang, I18N_STRINGS[key]["zh-TW"])
```

用戶可在 profile 設定 `language = "vi"`,Bot 自動切語言。
