使用方式
  1. 下面 4 個測試帳號已預載(U_TEST_* 前綴),點「重新整理」可更新狀態
  2. 選 user + 輸入任意 groupId(例:G_001)→ 按「綁定」
  3. 觀察回應:HTTP code + reply_text(這是實際會送給用戶的 LINE 訊息)
  4. 每個 user 可依方案試綁到上限為止:paid_799 1 群 / paid_799_year 3 群
  5. paid_399 / trial 帳號 → 試綁 → 應被拒(資格不符)
  6. 用完按下方「重置測試資料」清回預設狀態
注意: U_TEST_* 前綴的帳號與群組都是測試用的,重置會清掉,但不會影響真實用戶。
